# recovery-access-gap-index
Geospatial public health dashboard mapping overdose burden and recovery access gaps across Massachusetts.
## Project Goal

This project investigates whether recovery, harm reduction, and treatment resources in Massachusetts are located where overdose burden and social vulnerability are highest.

The final output will be an interactive dashboard and short research brief identifying municipalities that may be underserved relative to overdose burden.

## Research Question

Which Massachusetts communities experience high overdose burden but limited geographic access to recovery and harm reduction resources?

## Planned Features

- Massachusetts municipality-level overdose burden map
- Recovery and harm reduction resource locator map
- Burden vs. access comparison
- Social vulnerability overlay
- Recovery Access Gap Index
- Priority municipality ranking
- Public health recommendation labels

## Data Sources

Planned public datasets include:

- Massachusetts overdose death data
- Massachusetts EMS opioid-related incident data
- MassGIS municipality boundaries
- Mass.gov harm reduction program locations
- Syringe service program locations
- Peer recovery support centers
- SAMHSA treatment locator data
- CDC/ATSDR Social Vulnerability Index
- ACS Census demographic data

## Methodology

This project will create three core municipality-level scores:

1. **Overdose Burden Index**
2. **Recovery Access Index**
3. **Social Vulnerability Index**

These will be combined into a final **Recovery Access Gap Score** to identify communities with high need and limited access.

## Intended Use

This project is designed as a public health decision-support tool for identifying communities that may warrant further investigation, outreach, mobile services, or recovery/harm reduction resource investment.

## Limitations

This analysis estimates geographic and structural access. It does not measure service quality, waitlists, capacity, hours of operation, stigma, insurance acceptance, or actual service utilization.
